## 제품 비전

- 핵심 가치: 일상 속 정서 회복을 돕는 한국형 3D 동물 정서 동반자(펫-컴패니언 페르소나)
- 차별점: 온디바이스 SER + 프라이버시, MCP 기반 멀티에이전트 코칭, Autopilot로 스켈레톤을 성과 기반 확장
- 목표 경험: 10–20분 대화와 3–5분 미니 리추얼로 안정/집중/수면/기분 개선
- 윤리/안전: 위험 신호 자동 감지 → 전문가 연결 옵션, 낙인 금지/저자극 톤

### 제품 목표(12개월)
- 세션 완주율 ≥ 60%, 기분 개선(↑) ≥ 40%
- 7일 리텐션 ≥ 30%, LLM비용 ≤ $5/인·월
- 3D 씬 안정 FPS(중급 단말 24–30fps+), 초기 로딩 3–5s 내

### 핵심 원칙
- 로컬 우선(프라이버시/지연), 안전 우선, 작은 배치/빠른 학습, 문화적 맥락 존중
- 고정: persona/style/safety/rubric/action_mapping, 유연: LLM 즉시 합성 + Autopilot 제안

### 주요 사용자 시나리오
- 퇴근 후 빠른 안정: 음성으로 불안을 토로 → 동물의 느린 접근, 호흡 리추얼 추천
- 공부 전 집중: 짧은 루틴(노이즈/미니멀 음악) 실행, 씬 동기화
- 잠들기 전 수면 연결: 저자극 앰비언트 + 부엉이 씬, 자동 밝기 조절
